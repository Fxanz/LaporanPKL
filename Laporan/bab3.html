PELAKSANAAN PRAKTEK KERJA INDUSTRI

  1. 3.1 Rencana Kegiatan

    1. Praktek Kerja Industri dilaksanakan di Telkom Diva area Denpasar pada bagian dengan rencana kegiatan saat Praktek Kerja Industri, antara lain:

      1. Menaati tata tertib perusahaan.
      2. Mempelajari system kerja perusahaan.
      3. Membantu karyawan yang lain.
      4. Melaksanakan sesuai yang di tugaskan
      5. Menerapkan ilmu yang telah dipelajari di sekolah.
      6. Menambah pengalaman di dunia industri.

    2. 3.2.2 Deskripsi tugas yang dilaksanakan.

      Berikut ini deskripsi tugas dari beberapa uraian tugas yang dikerjakan di Balitop Digima Solutions:

      1. • Membuat wordpress di localhost :

        1. Unduh file wordpress di wordpress.org
        2. Unduh xampp di apachefriends.org
        3. Buka xampp dan aktifkan Apache dan MySQL
        4. Extract file zip dari wordpress.org di dalam folder xampp => htdocs
        5. Buka browser dari ketik “localhost/phpmyadmin”
        6. Pergi ke bagian database, lalu beri nama database dan pilih “create”
        7. Pergi ke folder xampp => htdocs lalu cari file yang bernama “wp-config.php”
        8. Ubah define('DB_NAME', ‘(ketik nama database yang baru dibuah disini)’)
        9. Buka browser dari ketik “localhost/(nama file wordpress)”
        10. Buat konfigurasi wordpress (nama web, nama admin, password)
        11. Penginstallan wordpress sudah selesai
